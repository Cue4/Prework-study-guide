HTML

Learned the format of a !DOCTYPE HTML and the contents within. Includint but not limited to:

  • Head: The head element contains information about the webpage.
  • Body: The body element represents the visible contents shown to the user.
  • Main: The main element represents the main content of the body of a document or application.
  • Title: The title element defines the title of the webpage.
  • Division: The div element create a division in the page.
  • Span: The span element create an inline division.
  • Style: The style element defines the CSS styles associtated with the website.
  • Base: The element defines the base URL for a webpage.
  • Link: The link element connects an external resource to the HTML document.
  • Meta: The meta element defines the (charset)charcter set, description, keywords, author and viewport.
  • H2-H6: The h1-h6 elements contains what is renered on the page.
  • Sections: The section elements are semantic elements that define a section in a document that contains thematic content.
  • Img: The img element contains information about the images that are displayed.
  • Src: The src attribute defines the location of the image.
  • A Href: The a href contains an active link which will direct the viewer to a webpage.
  • Alt: The alt attribute defines an alternative text discription of the image.
  • Indetifier (id): The id attribute identify elements and select them.
  • Unordered List (ul): The ul element places a bullet point next to the (li)listed items.
  • Ordered List (ol): The ol element places a number next to the (li)listed items.
  • List Items (li): The li are a list of items that are numbered or in bullet form.
  • Paragraph (p): The p represents a paragraph or block text.
  • Break (br): The br element creates a line of empty space or a line break
  • Card: .card selector contains width, margin, border, and box-shadow properties.
  • Class: A class attribute allow us to share a CSS rule to any element we choose by assinging the rule to a class attribute with a class selector.

CSS

  • Margin: A margin indicates how much space we want around the oustide of an element.
  • Padding: A padding indicates how much space we want around the content inside an element.
